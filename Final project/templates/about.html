About portal

How to use this website

This portal helps students, faculty, coordinators, and organisers view events, register, check stats, and see announced results.

Login and signup

New student participants can create an account, and existing users can log in with their saved details.

Select event

Users can choose a school, open an event, check competition details, and register when registration is available.

Stats and winners

The stats page shows school-based rankings, and the winners page lets users search results with filters.

Live result board

When an event result is announced, users can open the event, choose a competition, and view the published result.

Profile section Users can update profile picture and change password from their profile page.
Event registration Students can register inside live events by choosing the required competition and filling their details.
Result viewing Use the View Result option inside an event to check competition-wise announced results.
